Плагин InterPut позволяет максимально эффективно использовать пространство сайта.

Как использовать?

HTML

  1. <div class="interPut-wrapper">
  2.   <label>Расстояние до МКАД</label>
  3.   <div class="interPut-item">
  4.     <div>
  5.       <input name="value[min]" type="text" placeholder="От" />
  6.     </div>
  7.     <div class="separator"></div>
  8.     <div>
  9.       <input name="value[max]" type="text" placeholder="До" />
  10.     </div>
  11.     <div class="suffix">км</div>
  12.   </div>
  13. </div>

JS

  1. $('.interPut-wrapper').interPut();

Опции

checkMin

Значение MIN должно быть строго меньше значения MAX

  1. По умолчанию: true
  2. Опции: boolean (true / false)

maxNumInput

Максимально вводимое значение

  1. По умолчанию: 1E9
  2. Опции: integer

valueTitle

Title поля InterPut при наведении (хорошо работает с tooltip плагинами)

  1. По умолчанию: "Нажмите, чтобы выбрать интервал"
  2. Опции: string

inputMin

Используется для поиска поля со значением MIN

  1. По умолчанию: "input[name*='min']"
  2. Опции: string

inputMax

Используется для поиска поля со значением MAX

  1. По умолчанию: "input[name*='max']"
  2. Опции: string
